# Starter Investigator Picks

← [[games/arkham-lcg/index|Back to Arkham LCG]]

---

The Core Set (2026) ships five pre-built investigators — a clean starting roster:

| Investigator | Class | First-game feel |
|---|---|---|
| Daniela Reyes | Guardian | Straightforward fighter, soaks damage |
| Joe Diamond | Seeker | Reliable clue engine, card draw |
| Isabelle "Izzie" Barnes | Survivor/Mystic | Flexible; fight + protective spells |
| Trish Scarborough | Rogue | Evasion-focused, slippery |
| Dexter Drake | Mystic | Stage magician, willpower/spell-focused |

## Building a first deck
- Start from the **pre-built deck** for your investigator.
- Tweak using cards from the Core Set as you learn.
- See [[games/arkham-lcg/notes/tips|Tips]] for deckbuilding guidance.
