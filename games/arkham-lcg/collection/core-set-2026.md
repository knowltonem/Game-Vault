# Core Set (2026) — "Chapter Two"

← [[games/arkham-lcg/collection/my-collection|Back to collection]]

---

> The current evergreen base game and my **starting point**. Released March 20, 2026 (SKU **AHC100**, MSRP **$69.99**). Launches the game's **"Chapter Two"** — both a fresh on-ramp and a continuation, set months after a calamity strikes Arkham. It **replaces the Revised Core Set (2021)** as the evergreen core, while staying fully compatible with all prior Arkham products.

## Why this is the starting point
- **1–4 players out of one box** — no second core needed.
- New **3-scenario introductory campaign** (*Brethren of Ash*).
- **5 mechanically-new investigators**, each with a pre-built deck.
- Introduces the **Chapter Two / legacy** split: older cards are still playable, but future scenarios target the Chapter Two pool — a smaller, more balanced set to learn from.
- Extras: token bag, encounter-card dividers, location connection tokens.

## The five investigators
| Investigator | Class | Note |
|---|---|---|
| Isabelle "Izzie" Barnes | Survivor (Mystic lean) | New character; Jenny Barnes's sister, wields the .45s |
| Trish Scarborough | Rogue | Former spy; evasion-focused |
| Joe Diamond | Seeker | Private eye; straightforward card-draw engine |
| Daniela Reyes | Guardian | Combat-focused; changed least |
| Dexter Drake | Mystic | Stage magician returning from the original core; willpower/spell-focused |

## Notes
- Reviews praised production and investigators; mixed on *Brethren of Ash* (some found it underwhelming / brutally hard).
- 10th-anniversary "reboot" chapter; follows the events of the *Drowned City* campaign.
