## Deckbuilding Impact

**Fearless is level 2** — Greystoke cannot include it in a starting deck without spending XP. This affects the Greystoke deck list. Replace with level 0 Fearless from old Core Set OR replace with another level 0 Mystic skill.

**Guts, Overpower, Perception are now Neutral** — this makes them available to more investigators without splash restrictions. No impact on existing deck lists since both investigators already had access.
