# First-Game Setup

← [[games/arkham-lcg/index|Back to Arkham LCG]]

---

## Before your first game
1. Read the **Learn to Play** booklet; keep the Rules Reference for lookups.
2. Pick an investigator (a straightforward Guardian or Seeker is friendliest).
3. Use a **pre-built starter deck** for your investigator.
4. **Sleeve your cards** (see [[games/arkham-lcg/buying/sleeves|sleeves]]).

## Setting up a scenario
1. Build the **chaos bag** for campaign + difficulty.
2. Build the **encounter deck** from the sets the scenario lists.
3. Lay out **act** and **agenda** decks (1 on top).
4. Place starting **locations** per the scenario diagram.
5. Each investigator: place card, take signatures, set health/sanity, shuffle, draw 5 (mulligan once).
6. Resolve any **"Setup"** instructions.

## First-time tips
- Don't optimize — just play; the game teaches itself over a campaign.
- Standard is the intended baseline; Easy exists for a gentler intro.
- Expect to **lose scenarios** — that's story, not game-over.
- Keep a Campaign Log from scenario one.
