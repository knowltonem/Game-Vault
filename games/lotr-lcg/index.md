# Lord of the Rings: The Card Game (LCG)

> A cooperative Living Card Game for 1–4 players. Work together to complete quest scenarios in Middle-earth against an encounter deck controlled by the game. No PvP — beat it together or lose together.

**Publisher:** Fantasy Flight Games / Asmodee  
**Edition:** Revised Core Set (2022) — fully compatible with all original expansions
